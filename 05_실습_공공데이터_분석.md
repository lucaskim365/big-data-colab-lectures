# [빅데이터 분석] Part 5: 실습 - 공공데이터 분석 (EDA)

지금까지 배운 내용을 바탕으로, 실제 데이터셋인 **Titanic(타이타닉)** 데이터를 탐색하고 시각화하며 인사이트(Insight)를 도출해 봅니다.

---

## 1. 분석 준비 및 데이터 로드

```python
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 데이터 로드 (Seaborn 제공 데이터셋 활용)
df = sns.load_dataset('titanic')

# 데이터 구조 확인
print(df.head())
print(df.info())
```

---

## 2. 데이터 탐색 및 정제 (Cleaning)

데이터를 분석하기 전, 비어있는 값이나 데이터 형식을 정리합니다.

```python
# 결측치 확인
print(df.isnull().sum())

# 나이(age)의 결측치는 평균값으로 채우기
df['age'] = df['age'].fillna(df['age'].mean())

# 'embark_town'의 결측치는 가장 많이 나온 곳으로 채우기
most_freq = df['embark_town'].mode()[0]
df['embark_town'] = df['embark_town'].fillna(most_freq)

# 불필요한 열 삭제 (예: deck)
df.drop('deck', axis=1, inplace=True)
```

---

## 3. 핵심 데이터 분석 (Statistics)

생존율에 영향을 주는 요소들을 분석해 봅니다.

```python
# 성별에 따른 생존율
print(df.groupby('sex')['survived'].mean())

# 객실 등급(pclass)에 따른 생존율
print(df.groupby('pclass')['survived'].mean())
```

---

## 4. 시각화 피드백 (Dashboard)

그래프를 통해 더 명확한 인사이트를 얻습니다.

```python
# 성별에 따른 생존자 수 (countplot)
sns.countplot(data=df, x='sex', hue='survived')
plt.title('Survival Count by Sex')
plt.show()

# 객실 등급별 생존율 변화 (pointplot)
sns.pointplot(data=df, x='pclass', y='survived', hue='sex')
plt.title('Survival Rate by Pclass and Sex')
plt.show()

# 나이 분포 확인 (histplot)
sns.histplot(data=df, x='age', kde=True)
plt.title('Age Distribution')
plt.show()
```

---

## 5. 결론 및 인사이트 도출

데이터 분석을 통해 얻은 결과를 한 문장으로 정리해 봅니다.

- **결론 1**: 여성의 생존율이 남성에 비해 압도적으로 높았다 (여성 약 74%, 남성 약 18%).
- **결론 2**: 1등석 승객의 생존율이 가장 높았으며, 하위 클래스로 갈수록 생존율이 급격히 감소했다.
- **결론 3**: 어린이(유아기)의 생존 확률이 상대적으로 높게 나타났다.
